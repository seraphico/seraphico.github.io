python基础环境

俗话说，工欲善其事必先利其器,学习任何一门知识或者语言,首先要具备良好的基础环境

配置一个属于自己的编程环境是相当重要的。
环境：

  • 操作系统：Ubuntu 14.04.2 LTS
  • python软件：python2.7.9

  • ubuntu下面安装python需要注意：
  • sudo apt-get install zlib1g-dev libreadline-dev libcurl4-openssl-dev sqlite3 sqlite
